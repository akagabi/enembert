# enemBERT

Modelo aberto que encontra os 5 elementos da proposta de intervenção (Competência 5 do ENEM)
em redações — agente, ação, modo/meio, efeito e detalhamento — como spans no texto.

- **Não dá nota.** Aponta elementos presentes e ausentes, apontando as palavras exatas.
- Roda no navegador (transformers.js) — a redação não sai do seu computador.
- Projeto independente, não afiliado ao INEP/MEC.

Status: em construção. Ver `docs/design.md`.
